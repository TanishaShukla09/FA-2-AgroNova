# AgroNova - Smart Farming Assistant

> AI-Powered Agricultural Intelligence for Farmers Worldwide

AgroNova is a multilingual smart farming assistant built with **Streamlit** and powered by **Google Gemini AI**. It provides personalized crop recommendations, pest diagnosis, seasonal crop calendars, weather alerts, and expert farming advice tailored to your country and language.

---

## Features

- **AI Farming Assistant** - Ask any farming question and get expert, region-specific advice
- **Multi-Country Support** - Pilot countries: Canada, India, Ghana (+ 7 more)
- **Multilingual Interface** - Supports English, Hindi, French, Tamil, Telugu, Bengali, Spanish, and more
- **Seasonal Crop Calendar** - Planting and harvesting schedules by season
- **Pest and Disease Alerts** - Contextual warnings about pests, rainfall, and temperature changes
- **Query History** - Browse, re-ask, or delete past AI conversations
- **Feedback System** - Star ratings, feedback types, and a submission form
- **Dashboard Stats** - Track total queries, crop recommendations, and calendar views
